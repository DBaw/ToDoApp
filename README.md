# ToDoApp

**ToDoApp** is a simple and elegant TODO application built with **Avalonia UI**, designed to make managing your tasks and notes effortless. The app supports multiple users and provides essential CRUD operations for notes.

## Features

- **User Management**
  - Create new users
  - Login to existing accounts

- **Notes Management**
  - Create notes
  - Read notes
  - Update notes
  - Delete notes

- **Modern UI Design**
  - Built with Avalonia UI and styled using the Fluent theme

## Technical Details

This application is powered by **Entity Framework Core** for data persistence, using SQLite as the database engine. It also leverages **dependency injection** via Microsoft's Extensions library for clean and maintainable code.

## Dependencies

The following dependencies were utilized in building this application (beyond the default Avalonia Windows project setup):

- **CommunityToolkit.Mvvm**
- **Material.Avalonia**
- **Microsoft.EntityFrameworkCore**
- **Microsoft.EntityFrameworkCore.Relational**
- **Microsoft.EntityFrameworkCore.Sqlite**
- **Microsoft.EntityFrameworkCore.Tools**
- **Microsoft.Extensions.DependencyInjection**
- **Newtonsoft.Json**
